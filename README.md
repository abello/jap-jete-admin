# Setup

1. Instalo `virtualenv` dhe krijo nje virtual environment
    
    `pip install virtualenv`
    
    `virtualenv venv`

2. Aktivizo virtual environment
    
    `source venv/bin/activate`

3. Instalo dependencies
    
    `pip install -r requirements.txt`

4. Nis serverin
    
    `python server.py`
